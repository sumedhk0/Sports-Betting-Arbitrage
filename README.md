# Sports-Betting-Arbitrage
Sports Betting Arbitrage Calculator. Uses Python and API from The Odds API.

TO USE:
Go to https://the-odds-api.com/#get-access and get your API key.

Create a new file called .env in the project.

Add the following line (without the parentheses):

API_KEY=(your API key)

If you want to use multiple API keys for more searches, add the following instead:

API_KEYS=(your API keys, comma separated, no spaces, one 1 line, no quotes)

Run and get your arbitrage opportunities!
